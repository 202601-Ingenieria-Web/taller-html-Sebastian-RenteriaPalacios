# Taller 1 - PokeAPI Web

Aplicacion web sencilla construida con HTML, CSS y JavaScript que consume la API publica de PokeAPI para buscar un Pokemon por nombre y mostrar su informacion principal.

## Proposito del proyecto

Este proyecto fue desarrollado como parte del Taller 1 de Ingenieria Web para practicar:

- Estructura basica de una pagina web con HTML.
- Estilos y maquetacion con CSS.
- Consumo de APIs REST desde JavaScript usando `fetch`.

## Caracteristicas

- Busqueda de Pokemon por nombre (por ejemplo: `ditto`, `pikachu`, `charizard`).
- Consulta a `https://pokeapi.co/api/v2/pokemon/{nombre}`.
- Visualizacion de:
	- Imagen del Pokemon.
	- Nombre e ID.
	- Tipos.
	- Peso y altura.
	- Estadisticas base.

## Estructura del proyecto

- `PokeAPI.html`: estructura principal de la pagina y logica JavaScript.
- `styles.css`: estilos visuales del contenido y la presentacion.
- `poke_bg.webp`: imagen de fondo.

## Como ejecutarlo

No requiere instalacion de dependencias ni servidor de backend.

1. Clona el repositorio:

```bash
git clone https://github.com/202601-Ingenieria-Web/taller-html-Sebastian-RenteriaPalacios.git
```

2. Entra a la carpeta del proyecto:

```bash
cd taller-html-Sebastian-RenteriaPalacios
```

3. Abre `PokeAPI.html` en tu navegador.

## Uso

1. Escribe el nombre del Pokemon en el campo de busqueda.
2. Presiona el boton **Ingresar**.
3. Revisa la informacion mostrada en pantalla.
